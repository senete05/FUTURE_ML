## 🎯 WHAT EACH FILE DOES

### ⭐ ESSENTIAL CODE FILES

#### 1️⃣ `ticket_classification_system.py` (20 KB)
**The Main System - Everything in One File**

Contains all 6 sections:
- Load & explore data
- Preprocess text
- Extract features (TF-IDF)
- Train category classifier
- Train priority predictor
- Make predictions

**Usage:**
```bash
python ticket_classification_system.py
```

**Output:**
- Trains both models
- Shows accuracy, precision, recall
- Saves 4 model files
- Prints example predictions

---

#### 2️⃣ `inference.py` (7 KB)
**Make Predictions on New Tickets**

Usage:
```bash
# Single ticket
python inference.py --text "I can't login"

# From CSV file
python inference.py --input new_tickets.csv --output results.csv
```

**Output:**
- Category prediction
- Priority prediction
- Confidence scores
